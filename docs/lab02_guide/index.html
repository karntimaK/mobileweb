ศูนย์ศิลปวัฒนธรรม

หอศิลป์ประจำมหาวิทยาลัยขอนแก่น ตั้งอยู่บริเวณริมบึงสีฐานฝั่งตะวันตก ถนนศรีฐาน ตำบลศิลา อำเภอเมืองขอนแก่น จังหวัดขอนแก่น เป็นหอศิลป์แห่งแรกและหอศิลป์ที่ใหญ่ที่สุดในจังหวัดขอนแก่น

หอสมุดกลาง มหาวิทยาลัยขอนแก่น

เปิดบริการนั่งอ่านในพื้นที่ทุกวัน
